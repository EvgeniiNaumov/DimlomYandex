# ScooterProject

Автотест для проверки создания и получения заказа в системе Яндекс Самокат.

## Описание

1. Создаёт заказ через API.
2. Проверяет, что заказ можно получить по его трек-номеру.

## Запуск тестов

```bash
pytest test_create_and_get_order.py -v
```
